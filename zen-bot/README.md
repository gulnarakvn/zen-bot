# ZenFlow — Автопостинг Яндекс Дзен

Автоматическая генерация и публикация контента в Яндекс Дзен через RSS.

## Структура

```
zen-bot/
├── main.py          — FastAPI приложение + планировщик
├── generator.py     — Генерация текстов через Claude API
├── rss_builder.py   — Создание RSS файлов
├── templates/
│   └── index.html   — Веб-панель управления
├── rss/             — Сгенерированные RSS файлы (авто)
├── channels.json    — База каналов (авто)
├── requirements.txt
└── Procfile
```

## Деплой на Railway

### 1. Загрузить на GitHub
```bash
git init
git add .
git commit -m "init zenflow"
git remote add origin https://github.com/ВАШ_АККАУНТ/zen-bot.git
git push -u origin main
```

### 2. Создать сервис на Railway
- railway.app → New Project → Deploy from GitHub repo
- Выбрать репозиторий zen-bot

### 3. Добавить переменную окружения
```
ANTHROPIC_API_KEY = ваш_ключ
```

### 4. Готово!
Панель управления откроется по адресу вашего Railway проекта.

## Подключение к Яндекс Дзен

1. Создать аккаунт на dzen.ru
2. Перейти в настройки канала → Источники → RSS
3. Вставить ссылку: `https://ВАШ_ДОМЕН.railway.app/rss/SLUG_КАНАЛА.xml`
4. Дзен будет проверять RSS каждые несколько часов

## Как работает

- Каждые 4 часа скрипт проверяет каналы
- Если лимит постов на день не достигнут — генерирует новый пост
- Пост добавляется в RSS файл
- Яндекс Дзен подтягивает RSS и публикует

## Веб-панель

Доступна по корневому URL Railway проекта:
- Добавить/редактировать/удалить канал
- Ручная генерация поста (кнопка ▶)
- Статистика постов
- Ссылка на RSS для подключения к Дзен
